Kode CPU Configuration

Penggunaan dan Deskripsi Kode

Kode ini digunakan untuk mengatur konfigurasi CPU pada sistem. Berikut adalah fungsi-fungsi yang tersedia: 

        cpus - mendapatkan CPU yang tersedia
        online - [01]+ - membalik status inti CPU
        reset - [policy] - mengatur ulang semua frekuensi maksimum dan minimum inti CPU dengan kebijakan CPU
        freqs - [0-9]+ - frekuensi yang tersedia untuk CPU yang diberikan
        cur - [0-9]+ - frekuensi saat ini untuk CPU yang diberikan
        max - [0-9]+ - frekuensi maksimum saat ini untuk CPU yang diberikan
        min - [0-9]+ - frekuensi minimum saat ini untuk CPU yang diberikan
        getp - [0-9]+ - mendapatkan kebijakan saat ini untuk CPU yang diberikan
        setf - [0-9]+, [frekuensi] - mengatur frekuensi untuk CPU yang diberikan (userspace)
        setmax - [0-9]+, [frekuensi] - mengatur frekuensi maksimum untuk CPU yang diberikan
        setmin - [0-9]+, [frekuensi] - mengatur frekuensi minimum untuk CPU yang diberikan
        setp - [0-9]+, [policy]

Contoh Penggunaan

Untuk mendapatkan CPU yang tersedia:

cpus

Untuk membalikkan status inti CPU menjadi online:

online [01]+

Untuk mengatur ulang semua frekuensi maksimum dan minimum inti CPU dengan kebijakan CPU:

reset [policy]

Untuk mendapatkan frekuensi yang tersedia untuk CPU tertentu:

freqs [0-9]+

Untuk mendapatkan frekuensi saat ini untuk CPU tertentu:

cur [0-9]+

Untuk mendapatkan frekuensi maksimum saat ini untuk CPU tertentu:

max [0-9]+

Untuk mendapatkan frekuensi minimum saat ini untuk CPU tertentu:

min [0-9]+

Untuk mendapatkan kebijakan saat ini untuk CPU tertentu:

getp [0-9]+

Untuk mengatur frekuensi untuk CPU tertentu (userspace):

setf [0-9]+, [frekuensi]

Untuk mengatur frekuensi maksimum untuk CPU tertentu:

setmax [0-9]+, [frekuensi]

Untuk mengatur frekuensi minimum untuk CPU tertentu:

setmin [0-9]+, [frekuensi]

Untuk mengatur kebijakan untuk CPU tertentu:

setp [0-9]+, [policy]
